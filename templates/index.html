Video downloader

Paste any video URL to download it. Powered by yt-dlp — supports YouTube, TikTok, Instagram, Vimeo, Twitter/X and 1000+ more.

Supported sites

YouTube TikTok Instagram Twitter / X Vimeo Facebook Reddit Twitch SoundCloud + 1000 more

Run locally

  1. 1Install dependencies
pip install -r requirements.txt
  1. 2Start the server
uvicorn main:app --reload
  1. 3Open http://localhost:8000 in your browser

Deploy to Render

  1. 1Push this project to a GitHub repository
  2. 2Go to render.com → New → Web Service
  3. 3Connect your repo — Render auto-reads render.yaml
  4. 4Click Deploy. Live in ~2 minutes.
